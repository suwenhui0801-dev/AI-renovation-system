# 墙面摆放、第一人称穿墙与视图留白修复说明

## 本次完成内容

1. 墙面摆放
- 后端 Furniture 数据增加 placement、wall、wall_offset、mount_height 字段。
- 壁灯、电视柜等支持 wallMount 的对象可以作为墙面对象放置。
- 2D 视图中，墙面对象以贴墙线段形式显示。
- 3D 视图中，墙面对象根据所属墙面自动贴墙、旋转并设置安装高度。
- 保存、导入、更新家具时会保留墙面相关字段。

2. 第一人称 3D 视角
- 新增进入第一人称按钮和编辑模式按钮。
- 使用 Three.js PointerLockControls。
- 支持鼠标控制方向，W/A/S/D 移动，ESC 退出。
- 第一人称移动采用“可穿墙但限制在整体户型外框内”的方案，方便直接进入其他房间展示，不会无限走出户型。
- 第一人称模式下不会触发原有拖拽编辑逻辑。

3. 视图留白修复
- 2D 与 3D 卡片改为弹性布局。
- 2D 网格画布和 3D 场景容器会自动撑满白色卡片剩余区域。
- 修复之前视图下方大面积空白的问题。
- resize 时按容器真实尺寸更新 canvas 和 renderer。

## 主要修改文件

- templates/index.html
- static/style.css
- static/app.js
- app.py

## 使用提示

- 选择支持上墙的对象后，点击 2D 房间边线附近或 3D 墙面即可放置到墙上。
- 进入第一人称后，鼠标查看，W/A/S/D 移动，ESC 退出。
- 当前第一人称支持穿过房间隔墙，但仍限制在整体户型范围附近。
