# 墙面高度自定义与方向/颜色修复说明

## 本次新增

1. 房间编辑器新增「墙面高度(m)」输入框，范围 2.2m - 6.0m。
2. 后端 Room 数据继续使用 `height` 字段，新增/应用/导入房间时都会保存该高度。
3. 3D 墙体高度改为读取 `room.height`，墙面挂载物高度会自动限制在当前房间墙高以内。
4. 2D 房间内增加「墙高 x.xm」文字提示，但 2D 不再用墙面颜色填充房间。

## Bug 修复

1. 修复单人床、双人床、双层床 2D 与 3D 方向不一致：将这三类 GLB 模型的 3D `rotationOffset` 统一改为 0。
2. 修复墙面颜色显示到地面的错误：
   - 3D 地面改为固定中性色；
   - 3D 墙体改为使用房间的 `wall_color`；
   - 2D 房间填充改为固定浅色，不再显示墙面颜色。
3. 删除房间编辑器里的「墙面材质」下拉框；前端不再提交墙面材质。
