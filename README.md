# 装修模拟游戏

一个基于 Flask 和前端技术的装修模拟游戏，支持 2D/3D 视图、语音指令和手势识别 

## 功能特点

- **多视图编辑**：支持 2D 平面视图和 3D 自由视图，可实时切换
- **房间管理**：创建、编辑、删除房间，设置房间尺寸、墙面材质和颜色
- **家具库**：丰富的家具类型，支持添加、编辑、删除家具
- **门窗管理**：为房间添加门和窗户，可自定义位置和尺寸
- **语音指令**：通过 DeepSeek API 支持语音输入指令
- **手势识别**：使用 MediaPipe 实现手势控制，支持抓取、移动、缩放家具
- **操作历史**：支持撤销/重做操作，方便修改和调整

## 技术栈

- **后端**：Python 3.7+, Flask 3.0+
- **前端**：HTML5, CSS3, JavaScript
- **3D 渲染**：Three.js (通过前端引入)
- **手势识别**：MediaPipe Hands
- **AI 语音处理**：DeepSeek API (通过火山方舟)

## 安装与运行

### 1. 克隆项目

```bash
git clone <repository-url>
cd renovation_game-11
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置 API Key (可选)

如果需要使用语音指令功能，需要配置火山方舟 API Key：

- 在 `app.py` 文件中修改 `ARK_API_KEY` 变量，或
- 设置环境变量 `ARK_API_KEY`

### 4. 运行项目

```bash
python app.py
```

然后在浏览器中访问 `http://localhost:5000` 即可开始使用 

## 使用指南

### 基本操作

1. **添加房间**：在侧边栏的「房间列表」中填写房间信息，点击「新增」按钮
2. **添加家具**：在「家具库」中选择家具类型，然后在 2D 或 3D 视图中点击放置
3. **添加门窗**：在「门窗列表」中填写门窗信息，选择所在墙面，点击「新增」按钮
4. **编辑对象**：选中对象后，在对应的编辑面板中修改属性，点击「应用」按钮
5. **删除对象**：选中对象后，点击「删除」按钮 删除家具时，也可以使用语音指令"删除{房间名}的{家具名}"来指定房间

### 语音指令

支持以下语音指令格式：

- 给{房间名}添加{家具名}
- 给{房间名}{左墙/右墙/上墙/下墙}添加门
- 给{房间名}添加窗
- 添加房间
- 撤销
- 重做
- 显示网格
- 隐藏网格
- 删除{家具名}
- 删除{房间名}的{家具名}
- 把{家具名}改成{颜色名}
- 把{房间名}的{家具名}改成{颜色名}
- 把{房间名}宽度改为{数字}米
- 把{房间名}深度改为{数字}米

### 手势控制

- **双手拉开/靠近**：调整家具大小
- **捏合开始**：抓取家具
- **捏合移动**：拖动家具
- **松开捏合**：放下家具
- **OK 手势**：完成当前操作
- **👎手势**：删除当前选中对象
- **☝️食指伸出移动**：移动光标
- **👍手势**：语音输入
- **"3"手势**：撤销上一步操作

## 注意事项

- 语音指令功能需要有效的火山方舟 API Key
- 手势识别功能需要浏览器支持摄像头访问
- 3D 视图可能在低配置设备上性能较低
